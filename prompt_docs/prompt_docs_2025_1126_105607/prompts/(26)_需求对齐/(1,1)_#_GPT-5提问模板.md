# GPT-5提问模板

### 核心原则

严格按照这些要求来操作。

### 拆分任务

1. 把任务拆成几个部分
2. 列出觉得模糊的地方
3. 制定分步骤的计划
4. 做好计划后，跟我确认计划是否OK

### 流程

1. 先搞清楚你懂了没有
2. 每完成一步给我看结果，等我检查
3. 所有步骤都做完之前不准结束

要是任务比较复杂，或者我说了“好好思考一下”这种句子的话，就用GPT-5思考模式。

###处理内容

目标：\[这里写你想要的结果]
限制条件：\[这里写清楚你的具体要求]
语调/风格/格式：\[比如要用大白话还是相对严谨的文风，回答要不要列一个表格或者写一个列表来按几点回答]
