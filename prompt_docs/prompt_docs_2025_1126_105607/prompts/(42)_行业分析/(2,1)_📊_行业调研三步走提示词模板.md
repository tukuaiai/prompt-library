📊 行业调研三步走提示词模板

第一步：行业地图提示词
目的：快速建立行业认知框架，了解全貌和基本规则

假如你是一位资深的行业分析师，请为我生成一份关于【某某行业】的全面入门指南。这份指南需要清晰简洁，并使用要点列表的格式，主要包含以下部分：

1. 核心定义与范畴：用一句话定义这个行业是做什么的，并说明其主要包含哪些细分领域

2. 市场规模与增长：提供当前全球或特定区域（如中国）的市场规模、年增长率，并预测未来3年的增长趋势

3. 产业链结构（价值链）：清晰描绘出行业的上游供应商、中游核心业务、下游分销和客户，并列出每个环节的代表性公司

4. 主要参与者：列出3-5家行业内的头部公司、创新公司或黑马公司，并简要说明他们各自的核心优势

5. 行业术语（行话）：解释10个对于理解该行业至关重要的专业词汇

第二步：商业模式和竞争格局深度剖析

目的：深入理解企业盈利逻辑、竞争程度和生存法则

在了解了【某某行业】的基础之后，请以战略顾问的视角为我深入分析其商业模式和竞争环境，内容应该包括：

1. 主流商业模式：详细拆解该行业内公司主要的盈利方式（例如硬件销售、订阅服务、广告、数据服务等），并举例说明

2. 目标客户画像：描述该行业的核心客户群体是谁（To B还是To C），他们的主要痛点和需求是什么

3. 核心产品和服务分类：将行业内的产品或服务进行分类，并说明各类产品或服务的价值主张和差异化

4. 波特五力模型分析：
   - 供应商议价能力
   - 购买者议价能力
   - 新进入者威胁
   - 替代品威胁
   - 行业内部竞争
   请对以上五种力量进行简要分析，评估该行业的整体吸引力

5. 主要挑战和壁垒：总结当前行业发展面临的最大挑战（如技术瓶颈、政策监管、成本问题等），以及新公司进入此行业需要克服的主要壁垒

第三步：未来趋势和个人机遇洞察

目的：展望未来，找到个人在未来格局中的位置


基于对【某某行业】现状和竞争格局的理解，请扮演一位富有远见的风险投资人（VC），为我展望其未来发展趋势，并为不同身份的个人提供可执行的建议。请包含以下内容：

1. 未来3-5年的关键趋势：预测未来几年内将深刻影响该行业的技术趋势、市场趋势和政策趋势

2. 颠覆性技术或模式：指出有哪些潜在的新技术、新模式可能会颠覆现有市场格局

3. 关键成功要素：总结未来在该行业中取得成功的公司需要具备哪些核心能力

4. 机遇与切入点：
   - 对于求职者：哪些职位和技能未来会非常抢手？进入这个行业的最佳路径是什么？
   - 对于投资者：哪些细分赛道具有投资潜力？评估投资标的时应重点关注哪些指标？
   - 对于创业者：存在哪些尚未被满足的市场需求（无人区）？有哪些创新切入点？

💡 使用指南

依次提问：按照提示词一→提示词二→提示词三的顺序向AI提问
替换关键词：将【某某行业】替换为你感兴趣的任何行业（如新能源汽车、跨境电商、短剧、IP运营等）
追问与互动：根据AI的回答，对感兴趣的点进行深入追问，例如：
"请详细解释一下XX技术"
"请对比A公司和B公司的商业模式有何不同"
"这个趋势对普通求职者意味着什么"

这套提示词框架遵循宏观→微观、现状→未来的逻辑，能帮助你在短时间内系统了解任何陌生行业，无论是求职、投资还是创业都能获得高价值的洞察。
