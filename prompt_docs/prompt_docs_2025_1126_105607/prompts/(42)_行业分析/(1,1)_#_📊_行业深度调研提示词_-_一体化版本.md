# 📊 行业深度调研提示词 - 一体化版本

假如你是一位同时具备行业分析师、战略顾问和风险投资人经验的专家，请为我生成一份关于【某某行业】的深度调研报告。这份报告需要系统全面，逻辑清晰，包含从宏观到微观、从现状到未来的完整分析框架。

## 第一部分：行业全景地图

请首先帮我建立对该行业的认知框架，包括：

1. 核心定义与范畴：用一句话定义这个行业是做什么的，并说明其主要包含哪些细分领域

2. 市场规模与增长：提供当前全球或特定区域（如中国）的市场规模、年增长率，并预测未来3年的增长趋势

3. 产业链结构（价值链）：清晰描绘出行业的上游供应商、中游核心业务、下游分销和客户，并列出每个环节的代表性公司

4. 主要参与者：列出3-5家行业内的头部公司、创新公司或黑马公司，并简要说明他们各自的核心优势

5. 行业术语（行话）：解释10个对于理解该行业至关重要的专业词汇

## 第二部分：商业模式与竞争格局

接下来请深入分析该行业的盈利逻辑和竞争环境：

6. 主流商业模式：详细拆解该行业内公司主要的盈利方式（例如硬件销售、订阅服务、广告、数据服务等），并举例说明

7. 目标客户画像：描述该行业的核心客户群体是谁（To B还是To C），他们的主要痛点和需求是什么

8. 核心产品和服务分类：将行业内的产品或服务进行分类，并说明各类产品或服务的价值主张和差异化

9. 波特五力模型分析：
   - 供应商议价能力
   - 购买者议价能力
   - 新进入者威胁
   - 替代品威胁
   - 行业内部竞争
   
   请对以上五种力量进行简要分析，评估该行业的整体吸引力

10. 主要挑战和壁垒：总结当前行业发展面临的最大挑战（如技术瓶颈、政策监管、成本问题等），以及新公司进入此行业需要克服的主要壁垒

## 第三部分：未来趋势与机遇洞察

最后请展望该行业的未来发展，并提供可执行的建议：

11. 未来3-5年的关键趋势：预测未来几年内将深刻影响该行业的技术趋势、市场趋势和政策趋势

12. 颠覆性技术或模式：指出有哪些潜在的新技术、新模式可能会颠覆现有市场格局

13. 关键成功要素：总结未来在该行业中取得成功的公司需要具备哪些核心能力

14. 机遇与切入点：
    - 对于求职者：哪些职位和技能未来会非常抢手？进入这个行业的最佳路径是什么？
    - 对于投资者：哪些细分赛道具有投资潜力？评估投资标的时应重点关注哪些指标？
    - 对于创业者：存在哪些尚未被满足的市场需求（无人区）？有哪些创新切入点？

💡 使用说明：
- 将【某某行业】替换为你感兴趣的具体行业名称（如新能源汽车、跨境电商、短剧、IP运营等）
- 可以要求AI对某个部分进行深入展开
- 建议根据回答进行追问，例如："请详细解释XX技术"、"对比A公司和B公司的差异"等
